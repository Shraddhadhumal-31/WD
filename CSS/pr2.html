Aim: Create a static web page using HTML

Introduction

A static web page is a simple web page written using only HTML. It does not change based on who views it or when it is viewed.

Why Use Static Pages?

Static pages are useful for sharing fixed information like resumes, company info, or instructions. They are fast, easy to create, and require no server-side processing.

Common Features

  • Simple to build and host
  • No interactivity with users
  • Same content every time
  • Useful for small websites or documentation

Conclusion

HTML allows us to build static pages quickly. These pages form the base of all websites and are great for learning web development basics.
