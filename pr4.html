Types of HTML Lists

Ordered List
A list where items appear in a numbered or lettered order using "ol tag".
Unordered List
A list where items appear with bullets using "ul tag" (disc, circle, square).
Description List
A list of terms and their definitions using "dl", "dt", and "dd" tag.
